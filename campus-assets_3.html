Setting up Notion

1
Create an integration

Go to notion.so/my-integrations → New integration → give it a name → copy the Internal Integration Token

2
Create two databases in Notion

One called Asset Registry and one called Inspections Log. See the schema guide below.

3
Share each database with your integration

Open the database → ··· menu → Connections → Add your integration

4
Copy the database IDs

Open the database as a full page → copy from the URL:
notion.so/DATABASE_ID?v=...
